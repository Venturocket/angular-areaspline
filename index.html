Angular-areaspline

A simple areaspline chart directive using d3's area chart.

Download this project as a .zip file Download this project as a tar.gz file

Example


Usage

Markup

As an element:

<areaspline
    data="{string}"
    width="{float}"
    height="{float}">
</areaspline>

As an attribute:

<div
    areaspline
    data="{string}"
    width="{float}"
    height="{float}">
</div>

Parameters

Param Type Details
data string Angular expression from which to get the data. Expects an array of numbers, e.g. [ 1, 2, 3, ...]
width integer Gets passed directly to the SVG
height integer Gets passed directly to the SVG
